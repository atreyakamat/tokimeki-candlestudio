DESIGN DOCUMENT — TOKIMEKI WEBSITE
1. Design Vision

The Tokimeki website should feel like stepping into a soft, cozy Japanese-inspired room filled with warmth, calm, and subtle beauty.

It should evoke:

A gentle emotional response
Visual comfort
A sense of quiet luxury

This is not a loud or flashy UI — it is delicate, breathable, and emotionally immersive.

2. Visual Identity System
🎨 Color Palette (Soft Pastel System)

Primary Colors:

Cream Beige: #F7EFE5
Soft Blush Pink: #F3DADF
Warm Rose: #C97C7C
Muted Maroon: #7A2E3A

Secondary Colors:

Sage Green: #DDE5DC
Dusty Lavender: #E6E0F2
Warm Brown: #8B5E3C

Neutrals:

Off White: #FBF9F6
Light Gray: #EAEAEA
Usage Rules:
Backgrounds → cream / off-white
Accents → blush + maroon
CTA → warm rose / muted maroon
3. Typography System
Font Pairing:
Heading: Playfair Display (or similar serif)
Body: Inter / Poppins (clean sans)
Styling:
Large, airy headings
Generous line-height
Soft letter spacing
Tone:

Elegant but not intimidating

4. UI Components (Design Language)
🧊 Cards
Rounded corners (16–24px)
Soft shadows (low opacity)
Slight hover lift
🔘 Buttons
Pill-shaped (rounded-full)
Soft gradients or solid pastel
Hover: slight scale + glow
🫧 Floating Elements
Small decorative shapes
Soft blur + opacity
Slow floating animation
🕯 Product Cards
Minimal layout
Image-first
Subtle hover zoom
5. Layout System
Grid:
Max width: 1200px
Center aligned
Generous padding (80–120px sections)
Spacing Philosophy:

Whitespace = luxury

6. Animation System
Motion Style:
Slow
Smooth
Natural
Animation Types:
Fade + Slide Up
Used for sections
Floating Animation
Subtle vertical movement
Hover Interactions
Scale: 1 → 1.05
Shadow increase
Scroll Reveal
Elements appear gradually
Tools:
Framer Motion

8. Cute Design Elements (IMPORTANT)

To enhance uniqueness:

🌸 Small sakura petals floating
✨ Subtle sparkle effects on hover
🕯 Candle glow effect (soft blur light)
💌 Handwritten-style accents

These should be subtle — not overwhelming.

9. Micro-Interactions
Button hover glow
Image hover zoom
Smooth page transitions
Cursor sparkle (optional)
10. Accessibility
Maintain contrast ratios
Readable font sizes
Smooth but not distracting animations
11. Final Design Principle

"Softness is the differentiator."

The website should feel:

Calm
Beautiful
Intentional

Every element must support the Tokimeki emotion — a gentle moment of happiness.
