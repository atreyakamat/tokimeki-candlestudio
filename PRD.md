TOKIMEKI Candle Studio — Website Product Requirements Document (PRD)
1. Product Overview

This PRD outlines the design and development of a visually immersive, emotionally engaging, multi-page website for Tokimeki Candle Studio.

The goal is to translate the brand’s core philosophy — “a gentle flutter of happiness” — into a digital experience that feels calm, aesthetic, and deeply expressive.

This is not a typical e-commerce website. It is a digital extension of a lifestyle brand.

2. Objectives
Primary Goals:
Build a premium, aesthetic, Japan-inspired website
Increase brand credibility beyond Instagram
Enable seamless browsing and product discovery
Convert visitors into buyers (direct or via WhatsApp)
Secondary Goals:
Establish strong brand identity
Improve bulk order inquiries
Create a memorable, shareable experience
3. Target Audience
Age: 18–35
Aesthetic-driven users (Pinterest / Instagram heavy)
Gift buyers
Lifestyle-focused individuals
People who value calm, cozy environments
4. Design Philosophy
Core Theme:

Japanese minimalism + soft, cozy aesthetics

Visual Direction:
Soft beige, pastel pink, warm brown tones
Subtle gradients and shadows
Clean serif + modern sans-serif typography
Soft animations (fade, float, blur)
Emotional Tone:
Calm
Gentle
Warm
Dreamy
UI Style Keywords:
“Soft luxury”
“Floating UI”
“Cute but premium”
5. Core UX Principles
Minimal friction
Emotion-first design
Scroll-based storytelling
Visual hierarchy driven by whitespace
Smooth transitions and micro-interactions
6. Site Architecture (5 Pages)
1. Home Page

Purpose: First impression + storytelling

Sections:

Hero section (tagline + soft animation)
Featured products
Brand story snippet
Categories (candles, sachets, etc.)
Testimonials
CTA (Shop / WhatsApp)
2. Shop Page

Purpose: Product browsing

Features:

Grid layout (clean, spaced cards)
Filters (type, price, occasion)
Hover animations (lift, glow)
Quick view modal
3. Product Detail Page

Purpose: Conversion

Sections:

Large product visuals
Description + scent notes
Emotional storytelling (“how it feels”)
Price + CTA (Buy / WhatsApp)
Related products
4. About Page

Purpose: Brand depth

Sections:

Meaning of Tokimeki
Founder story
Inspiration (Japanese calm living)
Behind the scenes (handcrafted process)
5. Contact / Bulk Orders Page

Purpose: Lead generation

Features:

WhatsApp CTA
Bulk order form
Social links
FAQ section

14. Final Vision

The website should feel like entering a calm, softly lit Japanese room filled with warmth and subtle beauty.

Every scroll should feel like discovering a small moment of happiness — a true “Tokimeki” experience.

15. Developer + Designer Prompt

“Create a visually immersive, soft, Japan-inspired website with floating UI elements, smooth animations, and a calm color palette. Focus on emotional storytelling, minimal design, and premium feel. Ensure the experience is smooth, modern, and delightful across all devices.”
